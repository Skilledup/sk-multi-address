Enhance WooCommerce with multiple address management - lets customers save, edit and select from multiple shipping/billing addresses during checkout.

== Description ==

SK Multiple Addresses for WooCommerce adds multiple address management capabilities to your WooCommerce store. Let customers save and manage multiple addresses for faster checkout.

= Key Features =

* Multiple address management (save, edit, delete)
* Quick address selection at checkout
* Custom naming for addresses (Home, Work, etc.)
* Seamless WooCommerce integration
* Mobile-friendly interface
* AJAX-powered for smooth interactions
* Full multilingual support
* Arabic/Persian number conversion support

= Requirements =

* WordPress 5.0 or higher
* WooCommerce 3.0 or higher
* PHP 7.2 or higher


== Frequently Asked Questions ==

= How can customers add multiple addresses? =

Customers can add addresses from their My Account > My Addresses page or during checkout.

= Is this plugin compatible with WPML/Polylang? =

Yes, the plugin is fully translatable and includes Persian (Farsi) translations out of the box.

= Can I customize which address fields are shown? =

Yes, administrators can enable/disable optional address fields from WooCommerce > Multiple Addresses settings.
